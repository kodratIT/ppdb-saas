| Constraint | Impact | Mitigation |
|-------------|--------|------------|
| Cloudflare Workers memory limit (128MB) | Code optimization required | Use D1/KV for heavy operations |
| Neon PostgreSQL connection limit | Concurrent users limited | Connection pooling, read replicas |
| R2 storage cost | File size limits | Compression, auto-deletion |
| Midtrans transaction fees | Revenue reduced | Build into pricing model |
